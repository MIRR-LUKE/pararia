# UI: 辞書管理画面を作る

## 状態

- 未着手
- GitHub Issue: `#39`
- 最終更新: `2026-03-27`

## 何のための画面か

学校名、教材名、模試名、講師名などの辞書を育てて、文字起こしと固有名詞推測の精度を上げる。

## この画面でやること

- 辞書一覧を見る
- 追加する
- 編集する
- 削除する
- どの範囲で使うか決める

## 画面に置くもの

### 一覧

- 正式名称
- 別名
- 種別
  - 例: 学校 / 模試 / 教材 / 講師 / 生徒
- 適用範囲
  - 例: 組織全体 / 特定生徒 / 特定講師
- 外部 STT に送るかどうか

### 編集

- 正式名称
- 別名一覧
- 種別
- 範囲
- 外部送信可否

## 大事なこと

- 辞書運用が現場で回ること
- 生徒名などの扱いが危なくならないこと
- 管理者が育てやすいこと

## 完了条件

- 辞書の追加、編集、削除が迷わない
- どの語が外部送信対象かを見分けられる
- 運用担当が画面だけで辞書を育てられる
