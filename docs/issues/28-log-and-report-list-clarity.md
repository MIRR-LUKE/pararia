# UI: ログ一覧とレポート一覧を、もっと見やすく整理する

## 状態

- 未着手
- GitHub Issue: `#41`
- 最終更新: `2026-03-27`

## 何のための画面か

生徒詳細の 3 タブを、時系列で見返しやすく、探しやすくする。

## 追加したいこと

### 面談ログ一覧

- review 状態
- 信頼状態
- 使われたレポート件数
- 次回確認あり

### 指導報告一覧

- チェックイン / チェックアウトの揃い具合
- 自動生成完了状況
- review 状態

### 保護者レポート一覧

- 共有状態
- 共有履歴
- 参照ログ件数

## 大事なこと

- 一覧の段階で状態が分かること
- 開かないと分からない情報を減らすこと
- 作業の戻り先として使いやすいこと

## 完了条件

- 一覧だけで次に開くべき項目を判断しやすい
- 面談ログ、指導報告、保護者レポートで状態表示の考え方がそろう
- 過去の履歴を探す時間が減る
