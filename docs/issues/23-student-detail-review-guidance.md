# UI: 生徒詳細に「要確認」の導線を足す

## 状態

- 実装済み
- GitHub Issue: `#36`
- 最終更新: `2026-04-13`

## 何のための画面か

生徒詳細の中で、いま何を確認すべきかをひと目で分かるようにする。

## この画面でやること

- 録音する
- ログを見る
- レポートを作る
- その前に、確認が必要なものに気づく

## 追加するもの

### 生徒詳細の上部か中段

- 要確認の文字起こし件数
- 確認待ちの固有名詞件数
- 再生成が必要なログ件数
- 送る前に見直すべきレポート件数

### ボタン

- `文字起こしを確認する`
- `ログを見直す`
- `レポートを確認する`

## 表示ルール

- 何もなければ静かでよい
- 要確認がある時だけ強く出す
- 録音や生成より先に、確認が必要ならそちらを前に出す

## 大事なこと

- 生徒詳細を開いた瞬間に次に何をやるかが分かる
- 録音だけして終わらない
- review が裏に埋もれない

## 完了条件

- 要確認があるときに生徒詳細で必ず気づける
- 次に押すべき導線が 1 つ以上見える
- 要確認がないときは静かで、通常作業の邪魔にならない
